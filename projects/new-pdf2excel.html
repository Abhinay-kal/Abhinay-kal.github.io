New PDF2Excel

OCR-driven transformation pipeline that converts noisy PDFs into structured spreadsheet-ready output.

View Repository Open Resume Contact

Overview

This project extracts data from scanned and layout-variant PDFs, then structures it into spreadsheet output for operational use. The focus is on accuracy and repeatability for messy source files.

Impact

  • Automated manual extraction flow from scanned documents.
  • Improved data usability with structured outputs suitable for analysis.
  • Reduced turnaround time on document-heavy workflows.

Tech Stack

Python OpenCV Tesseract OCR

Architecture

  • Preprocessing stage to improve text extraction quality.
  • OCR pass with post-processing for field normalization.
  • Export layer for table-mapped spreadsheet output.

Deliverables

  • Document ingestion and preprocessing scripts.
  • OCR extraction with cleaning and structuring steps.
  • Spreadsheet export pipeline ready for bulk operations.
